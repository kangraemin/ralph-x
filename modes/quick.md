# Quick Mode

Jump straight into coding. No planning phase.

## Pipeline
1. **Code** — Start implementing immediately
2. **Test** — Run tests, fix failures
3. **Iterate** — Repeat until done

## Behavior
- Skip all planning and design phases
- Write code first, fix issues as they come
- Each iteration: check what's broken → fix it
- Best for: small tasks, prototypes, well-understood problems
